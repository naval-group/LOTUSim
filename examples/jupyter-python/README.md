## Using lotusim with jupyter notebook

1. Run lotusim

    ```shell
    lotusim run
    ```

2. source environment

    ```shell
    source ${LOTUSIM_WS}/install/setup.bash
    ```

3. Run jupyter notebook

    ```shell
    jupyter-notebook ${LOTUSIM_PATH}
    ```
